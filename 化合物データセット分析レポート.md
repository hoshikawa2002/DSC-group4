## 📚 参考：専門用語解説

### SMILES記法とは？
- **Simplified Molecular Input Line Entry System**
- 化学構造を文字列で表現する方法
- 例: `CCO` = エタノール（お酒の主成分）

### 阻害値とは？
- 薬物が標的タンパク質の働きを止める強さ
- 低い値 = 強い効果 = 少量で効く
- 高い値 = 弱い効果 = 大量必要

### ベータラクタム系抗生物質とは？
- ペニシリンやセファロスポリンの総称
- 細菌の細胞壁合成を阻害して細菌を殺す
- 現代医療で最も重要な抗生物質群
